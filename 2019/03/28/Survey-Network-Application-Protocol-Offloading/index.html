  • Eiffel
    Leverage the O(1) FindFirstSet operation provided by modern CPU, design
    a data structure Integer Priority Queue, which groups packets with priority
    within a small range to the same bucket. The tree-structured bitmap is
    evaluated to accelerate software packet scheduling.

  • Loom

  • SENIC
    NetFPGA implementation of rate limiters (>10000), putting meta data onboard
    while packet data in host RAM. These rate limiters can use to realize many
    queueing algorithms (e.g. DRR, WFQ, WF2Q, HTB) in high-speed networks.

  • P4

  • ServerSwitch
    A smartNIC design with broadcom chip embedded to the NIC. App developpers
    can modify ServerSwitch driver in the kernel to add new packet processing
    pipelines. Use examples such as source routing, QCN, Bcube, In-Network
    Caching.

notes

  • To minimize tail latency

    • case1 : N stable large flows with accidentally small bursts.
      The key contributor to tail latency is the highest queueing point illustrated
      in DCTCP paper figure 12, when all synchronized flows increase each cwnd by
      1 before the ECN is carried back.
      The possible ways to minimize this effect is to either increase cwnd with an
      amplitude inversely proportional to N (LDCP). Or desynchronize (PACK) them to
      let one flow’s peak arrives at another flow’s trough.

    • case 2: incast. excessive amount of small bursts overwhelm the buffer within
      one RTT. PFC should be triggered.
